## 🤝 Contributing

We welcome contributions! Please follow these guidelines:

1. **Quality over quantity** - Only submit tools that provide real AI functionality
2. **Verify links** - Ensure the tool is accessible and working
3. **Avoid duplicates** - Check existing entries before submitting
4. **Use proper formatting** - Follow the existing structure in `data/tools.json`

### How to Add a Tool

1. Fork this repository
2. Add your tool to `data/tools.json` with:
   ```json
   {
     "name": "Tool Name",
     "url": "https://github.com/example/tool",
     "description": "Brief description (1-2 lines)",
     "category": "Developer Tools",
     "tags": ["tag1", "tag2", "tag3"],
     "pricing": "Free|Freemium|Paid",
     "stars": 1000,
     "language": "Python",
     "added_date": "2026-04-02",
     "verified": true
   }
   ```
3. Update the README.md with your tool in the appropriate category
4. Submit a pull request
