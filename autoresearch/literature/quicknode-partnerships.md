# QuickNode Partnership Analysis

**Date:** 2026-04-19
**Source:** https://www.quicknode.com/

## Overview
QuickNode is an enterprise-grade blockchain platform for teams requiring fastest, most reliable infrastructure. "Build Without Compromise."

## Key Products
- Core RPC API (82+ chains, 135+ networks)
- Streams (real-time data, backfills)
- Webhooks (instant blockchain alerts)
- Dedicated Clusters (private, isolated infrastructure)
- IPFS gateways and pinning
- Marketplace for partner APIs
- Validator as a Service
- SQL Explorer for data queries

## Partnership Program Structure

### 1. **Preferred Partner Network**
- Formal ecosystem of add-ons and partner APIs
- Categories: Trading & DeFi, Security & Compliance, Data & Analytics
- One-click integrations through marketplace
- Co-marketing opportunities

### 2. **Referral Program**
- Revenue-sharing model for referring customers
- Incentive-based for partners
- Trackable referral links

### 3. **Pro Services**
- Custom integrations and solutions
- Enterprise-grade support
- Dedicated infrastructure for large customers

### 4. **Startup Program**
- Resources for early-stage companies
- Support credits (implied)
- Networking and visibility
