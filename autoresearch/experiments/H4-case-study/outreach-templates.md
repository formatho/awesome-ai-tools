## Customization Checklist

Before sending any template, customize these sections:

### Required Customizations
- [ ] **Partner Name:** Replace [Partner Name] with actual company name
- [ ] **Contact Name:** Replace [Name] with actual person's name
- [ ] **Specific Reason:** Add 1-2 sentences on why you picked this partner
- [ ] **Case Study Angle:** Reference 1-2 relevant case study concepts
- [ ] **Contact Info:** Add Formatho website, Twitter, LinkedIn

### Optional Customizations (improves response rate)
- [ ] **Personal Connection:** Mention how you found them (LinkedIn, event, referral)
- [ ] **Product Detail:** Reference specific product/feature they offer
- [ ] **Mutual Customer:** If you have one, mention it (gold mine!)
- [ ] **Recent Content:** Reference a recent blog post or announcement from them
- [ ] **Community Ties:** Mention if you're in the same Discord/Slack community

### Pre-Send Validation
- [ ] Did I research the contact person before reaching out?
- [ ] Is the email/DM personalized beyond just name/company?
- [ ] Did I include a clear call-to-action (call or email reply)?
- [ ] Is the value proposition clear (3-5 leads, minimal effort)?
- [ ] Did I check for typos or formatting issues?

## Tracking and Attribution Setup

### UTM Parameter Structure

All links in the case study should include UTM parameters for tracking:

```
https://partner.com/pricing?utm_source=formatho&utm_medium=case_study&utm_campaign=joint_case_study&utm_content=[partner_name]
```

**Parameters:**
- `utm_source=formatho` — identifies Formatho as the source
- `utm_medium=case_study` — identifies content type
- `utm_campaign=joint_case_study` — identifies the campaign
- `utm_content=[partner_name]` — identifies specific partner (e.g., quicknode, alchemy)

### Tracking Setup Checklist

**Formatho Side:**
- [ ] Google Analytics configured with UTM tracking
- [ ] Custom event tracking for case study page views
- [ ] Lead capture form with source attribution
- [ ] Dashboard for monitoring traffic and conversions

**Partner Side (request access):**
- [ ] Google Analytics or equivalent configured
- [ ] Ability to add UTM parameters to links
- [ ] Lead capture form with source field
- [ ] Monthly performance report sharing

### Metrics to Track

**Traffic Metrics:**
- Page views by source (partner blog, Formatho blog, Reddit, Dev.to, etc.)
- Time on page (>2 min = engaged reader)
- Bounce rate (<60% = good content)
- Unique visitors vs. returning visitors

**Engagement Metrics:**
- Social shares (Twitter, LinkedIn, Reddit)
- Comments and discussion
- Backlinks and mentions
- Newsletter signups

**Lead Metrics:**
- Conversion rate (goal: 1-3%)
- Lead quality (enterprise vs. individual)
- Source attribution (which channel drove the most leads)
- Time to first contact

**Reporting:**
Weekly for first 2 weeks, then monthly.
